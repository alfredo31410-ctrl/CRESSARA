# Frontend de Cressara

Aplicación React construida con Vite y Tailwind CSS.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run preview
```

El servidor local utiliza `http://localhost:3000`.

## Organización

- `src/pages/public`: sitio público.
- `src/pages/legal`: aviso de privacidad y términos.
- `src/pages/admin`: acceso y panel administrativo.
- `src/components/layout`: navegación, pie y marca.
- `src/components/content`: bloques reutilizables de contenido.
- `src/components/ui`: primitivas visuales utilizadas por el panel.
- `src/features/auth`: estado y protección de autenticación.
- `src/services`: cliente API y contacto.
- `public`: recursos estáticos.
